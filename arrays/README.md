## Arrays

## WHEN TO USE ARRAYS

- When you need order
- When you need fast access / insertion and removal (sort of....)

## Big O of Arrays

| Operation | Big O         |
| --------- | ------------- |
| Insertion | It depends... |
| Removal   | It depends..  |

## Big O of Array Operations

| Operation     | Big O         |
| ------------- | ------------- |
| push          | O(1)          |
| pop           | O(1)          |
| shift         | O(N)          |
| unshift       | O(N)          |
| concat        | O(N)          |
| slice         | O(N)          |
| splice        | O(N)          |
| sort          | O(N \* log N) |
| forEach/map   | O(N)          |
| filter/reduce | O(N)          |
